# Calculadora Estatística Inferencial

Aplicação frontend para a Parte 2 do projeto de Calculadora Estatística:
teste de hipóteses com cálculo de p-valor e modo de p-valor informado.

## Recursos

- Página inicial integrada para Estatística Descritiva e Inferencial.
- Teste Z para média com desvio padrão populacional conhecido.
- Teste T para média com desvio padrão populacional desconhecido.
- Teste Z para proporção.
- Modo de p-valor informado para exercícios didáticos.
- Exemplos prontos: Aula 9, Tarefa 8 e Iris Dataset.

## Comandos

```bash
npm install
npm run dev
npm run build
```

O link da calculadora descritiva está centralizado em `DESCRIPTIVE_CALCULATOR_URL`
em `src/App.jsx`.
